# 古のCGIゲームである箱庭諸島を5系PHPからGolangへフルリプレイスする
現状、配布されている箱庭諸島2のスクリプトが5系のPHPで動いており瀕死寸前のため、フルモダンなスクリプトへリプレイスし甦らせるプロジェクトを始動しました！

 使用条件
―――――――――――――――――――――――――――――――――――
 箱庭諸島2のスクリプトについては、自己責任で使用する限り、基本的には
 自由に利用してもらってかまいません。ただし、変更を加えた場合は、変更
 したものを配布するには、後述の「配布条件」に従って下さい。

―――――――――――――――――――――――――――――――――――
 配布条件
―――――――――――――――――――――――――――――――――――
 箱庭諸島2 for PHPのスクリプトを改変し、それを他人に譲渡、配布する場合
 には、以下の制約を課します。

・無料配布であること。
・ゲーム画面のトップに表示される、スクリプトの配布元へのリンクを
  消すのを禁止すること。また、それ以外の改造は許可すること。
・本条件と同等に、改造したものの配布を許可すること。
・配布するページにおいて、オリジナルスクリプトの配布元として当サイトへ
  のリンクを置くこと。

 設置方法の説明
―――――――――――――――――――――――――――――――――――
――必要なファイル――
必要なファイルは以下の5つ、及び画像ファイルです。

config.php
hako-main.php
hako-html.php
hako-log.php
hako-turn.php
hako-mente.php
hako-mente-safemode.php


――ファイルの構成――
 
 実行可能なディレクトリ(755, 777)/
     |
     |-- hako-main.php
     |-- hako-html.php
     |-- hako-turn.php
     |-- hako-log.php
     |-- config.php
     |-- hako-mente.php
     |-- hako-mente-safemode.php
     |-- data         (生成される)
     |-- data.bak0〜3 (生成される)

 上記では、'data'とありますが、ディレクトリの名前は設定可能です。
 セキュリティのために、なるべく変更するようにして下さい。


――設置の手順――
'config.php'中の、「各種設定値」のところにある設定値を、適切な値に設定
します。

'hako-mente.php'を実行し、「新しいデータを作る」を実行します。

セーフモードがOnになっているサーバでは'hako-mente.php'ではデータファイル
を作成することができません。このような場合、'hako-mente-safemode.php'を
実行します。'hako-mente-safemode.php'ではdataディレクトリを自動で作成しま
せんので、あらかじめFTPソフトなどで作成しパーミッションを777などに変更して
おく必要があります。
また、config.phpの設定で$backupTimesは0にする必要があります。
  var $backupTimes	= 0;


'hako-main.php'を実行するとゲームが始まるはずです。
